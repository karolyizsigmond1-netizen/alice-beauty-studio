# Alice Beauty Studio — Demo Site

A modern, feminine demo site for **Alice Beauty Studio** (Budapest, Zugló) — a one-chair beauty studio specializing in bridal hair & makeup, balayage, keratin treatments, and occasion styling.

> Demo / portfolio piece — not officially affiliated with the original studio.
> Real studio: [alicebeautystudio.hu](https://www.alicebeautystudio.hu/)

## Stack

- Single-page static site — `index.html`, `styles.css`, `script.js`
- No build step, no dependencies
- Photography fetched from the original studio site (downloaded to `img/`)

## Design direction

**Refined-feminine maison** — cool blush base (decidedly not beige), deep aubergine ink, dusty rose accent. Typography pairs **DM Serif Display** (display) with **Instrument Serif** (italic accents), **Pinyon Script** (ornamental flourishes), and **Schibsted Grotesk** (body). Organic asymmetric border-radii on photo blocks, custom SVG botanical sprigs and a flower-of-life bloom motif appear throughout (logo halo, hero ornaments, testimonial crowns, footer rule, chatbot icon).

## Features

- **Hero** with orchestrated line-by-line title reveal, two-photo asymmetric portrait composition
- **Work / portfolio grid** with organic petal-shaped frames
- **Services list** — six-row hairline-divided table-of-contents style
- **About** section with sticky portrait, overlapping secondary photo, Pinyon-script "Alice" badge
- **Testimonials** in blush dome cards
- **Live booking section** with backlit photo and glass slots card
- **Demo appointment setter** (multi-step modal: service → date → time → contact → confirmation) — opens from any "Foglalás" button, service row, or live-slot tile
- **AI chatbot** ("Alice Asszisztens") — bottom-right bloom button, fully Alice-branded with scripted Hungarian responses covering services, pricing, hours, location, bridal, balayage, makeup, keratin, etc.
- Fully responsive

## Run

Just open `index.html` in any modern browser. No server required.
